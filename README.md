## 1팀의 활동

+ 1팀은 객체지향에 대해 이해와 git에 대한 숙련도를 위해 주1회 이상 스터디를 진행합니다.
+ 1팀은 서로의 코드를 리뷰하는 시간을 갖습니다.
+ 1팀은 팀원 모두 주차별 일정을 작성하여 계획적인 프로젝트를 진행합니다.
+ 각 스테이지를 분업하여 보다 효율적으로 프로젝트를 진행합니다.
